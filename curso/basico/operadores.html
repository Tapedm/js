Operadores e Procedencia

A procedencia vai de cima para baixo nesta ordem

  • () : Parentesis

Operadores aritimeticos :

  • ** : potência
  • * : Multiplicação
  • / : Divisão
  • % : Resto
  • + : Soma
  • - : Subtração

Operadores relacionais

Não tem regras para a ordem de procedencia, a ordem é lida da esquerda pra direita

  • > : Maior que
  • < : Menor que
  • >= : Maior ou igual a
  • <= : Menor ou igual a
  • == : Igual a
  • != : Diferente de (Comparação apenas de conteudo)
  • === : Identico a (Comparação de conteudo e tipo)

Operadores logicos

  • ! : Negação
  • && : AND
  • || : OR

Obs

Ternario :

Ternario são condições de verdadeiro ou falso e suas consequencias ( uma das formas de representar )

  • teste ? se for verdade rodar a : se for falso rodar b
  • 7 > 5 ? "Aprovado" : "Reprovado" = Aprovado
