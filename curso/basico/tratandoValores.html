Manipulando valores de vareaveis

Para transformar um dado pego do prompt em um tipo inteiro ou numero flutuante é só usar o comando :

Number.parseInt(window.prompt("Informe um numero"))
Number.parseFloat(window.prompt("Informe um numero"))
Number(window.prompt("Informe outro numero")) (Generalizado)

É necessario passar a variavel para um tipo numero pois se houver alguma operação matematica no caso a soma as variaveis vão apenas se cocatenar "juntando" na outra.

Obs:

O comando com Apenas o Number o proprio JavaScript defini um tipo apropriado para a variavel, mas em casos onde precisam de muita precisão é bom usar os mais especificos

Formatando numeros flutuantes :

para definir o numero de casas decimais ( no caso duas casas

variavel.tofixed(2)

Para substituir o ponto por uma virgula :

variavel.replace('.',',')
